## Robo4J-rpi-dof10imu-client
Robo4J Adafruit Adafruit 10-DOF IMU Breakout client example

The example is capable to handle data-points from Adafruit 10-DOF [Dof10Unit][] IMU Breakout unit and 
store them into the database for latter analysis.



properties values are according to the 'raspstill' util documentation

### Requirements:
1. clone and install robo4j project from Robo4J Github organisation
    * Robo4J will be installed into the local maven repository
    * command inside the project root folder: 
    ````
    $ gradle clean build install
    ````
